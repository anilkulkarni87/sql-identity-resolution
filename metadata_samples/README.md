# Metadata Samples

Load these CSVs into the corresponding `idr_meta` tables:

- `source_table.csv` -> `idr_meta.source_table`
- `rule.csv` -> `idr_meta.rule`
- `source.csv` -> `idr_meta.source`
- `survivorship_rule.csv` -> `idr_meta.survivorship_rule`
- `identifier_mapping.csv` -> `idr_meta.identifier_mapping`
- `entity_attribute_mapping.csv` -> `idr_meta.entity_attribute_mapping`
