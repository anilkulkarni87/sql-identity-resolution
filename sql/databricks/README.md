# Databricks Implementation

## Directory Structure

| Folder | Status | Description |
|--------|--------|-------------|
| **`core/`** | **Production** | Core IDR engine. Deploy these files to production. |
| **`testing/`** | **Development** | Scripts for testing, demos, and benchmarking. Do NOT deploy to production. |
| **`ops/`** | **Operations** | Helper utilities for metadata loading and maintenance. |

## Quick Start
Run `testing/IDR_QuickStart.py` notebook.
